# shadcn/ui

> shadcn/ui adalah kumpulan komponen yang dirancang dengan indah dan mudah diakses, serta platform distribusi kode. Dibangun dengan TypeScript, Tailwind CSS, dan primitif UI Radix. Mendukung berbagai framework termasuk Next.js, Vite, Remix, Astro, dan banyak lagi. Sumber Terbuka. Kode Terbuka. Siap untuk AI. Juga dilengkapi dengan alat baris perintah untuk menginstal dan mengelola komponen serta sistem registri untuk mempublikasikan dan mendistribusikan kode.

### Skema Registri

- [Skema Registri](https://ui.shadcn.com/schema/registry.json): Skema JSON untuk file indeks registri. Mendefinisikan struktur untuk kumpulan komponen, hook, halaman, dll. Membutuhkan array nama, beranda, dan item.
- [Skema Item Registri](https://ui.shadcn.com/schema/registry-item.json): Skema JSON untuk setiap item registri. Mendefinisikan komponen, hook, tema, dan kode terdistribusi lainnya dengan properti untuk dependensi, file, konfigurasi Tailwind, variabel CSS, dan banyak lagi.
